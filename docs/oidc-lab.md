# Part 2 - OIDC (OpenID Connect)

## Goal
Configure Auth0 as the Identity Provider and a SAML testing application as the Service Provider.

## What I configured
- SAML2 Web App addon in Auth0
- Assertion Consumer Service URL
- Entity ID / Audience
- IdP metadata XML
- X.509 certificate
- SAML login flow

## What I learned
SAML uses XML metadata, certificates, and assertions to establish trust between an Identity Provider and a Service Provider.
